### Xish AI Mobile (Expo React Native)

- Branding: Dark (#0A0A0A) with neon accents (#1DD1A1, #00F0FF)
- Font: Poppins
- Screens: Landing and Chat (SSE streaming + fallback)

Run:

```bash
npm install
npm run start
```

Config:

- Set `EXPO_PUBLIC_API_URL` in `app.config.js` or `.env` to backend URL
